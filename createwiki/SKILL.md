# 创建知识库 (createwiki)

## 触发条件

当用户说"新建知识库"并提供知识库名称时触发。

## 工作流程

1. **确定知识库名称**
   - 从用户输入中提取知识库名称
   - 如果用户未提供名称，提示用户输入

2. **创建目录结构**
   - 在 `C:/Users/Administrator/Desktop/知识库/<知识库名称>/` 下创建：
     - `raw/` — 原始资料目录
     - `raw/assets/` — 附件目录（图片等）
     - `wiki/` — 维基页面目录

3. **创建基础文件**
   - `AGENTS.md` — 架构说明文档（复制 default 的模板并修改）
   - `index.md` — 知识索引（初始化空框架）
   - `log.md` — 操作日志（记录创建事件）

4. **记录日志**
   - 在 log.md 中追加创建记录

5. **确认完成**
   - 向用户报告新知识库已创建
   - 提供新知识库的路径

## 目录模板

```
知识库/
  └── <名称>/
        ├── raw/
        │   └── assets/
        ├── wiki/
        ├── AGENTS.md
        ├── index.md
        └── log.md
```

## 注意事项

- 知识库名称可以包含中文
- 如果知识库已存在，询问用户是否覆盖
- 创建完成后，用户可以立即使用 Ingest 技能向其中摄入知识
